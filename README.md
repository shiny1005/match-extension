## ✨ Features

- **Auto Swipe** — Automatically likes (or skips) profiles based on your filters
- **AI Auto Chat** — Generates smart, natural opening messages and replies
- **Speed Control** — Adjustable delay to look human-like and avoid detection
- **Pause / Resume** with one click
- **Statistics** — Tracks swipes, matches, messages sent today
